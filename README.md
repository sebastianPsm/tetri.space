# tetri.space

## Dependencies:
- flask
- numpy
- windows-curses

## Start core with:
python3 core/core.py

## Start REST-api with:
~/workspace/tetri.space$ export FLASK_APP=rest.rest
~/workspace/tetri.space$ export FLASK_ENV=development
~/workspace/tetri.space$ flask run
